# 期中作业——NotePad PLus
## 一. 实验名称

## 二. 实验要求与目的
基本要求

1. NoteList中显示条目增加时间戳显示  
2. 添加笔记查询功能（根据标题查询）  

附加功能  
1. 界面美化
2. 可以更改记事本的背景颜色 
3. 可以更改记事本中字体的颜色大小
